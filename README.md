# whisperme-plan-hub

Living plan-of-record for the WhisperMe clinical Hebrew Speech-Language-Pathology project.

Open [`index.html`](./index.html) — it is the hub.

This repo contains:

- `index.html` — the dashboard: status, decisions log, phases, architecture, rules + lessons, findings library, open gaps.
- `research/` — six research reports (R1–R6) that ground every architectural decision in primary sources.

The hub is updated as the project's state changes. If the file is out of sync with what's actually true, the conversation is wrong.
